# Modern Java Full Stack Portfolio 🚀

A high-performance, premium React portfolio featuring a **"Pure Black + Colorful"** design system. This project showcases my expertise in Java Full Stack development, Computer Vision, and modern web design.

## ✨ Key Features

- **Pure Black + Colorful Theme**: A stunning high-contrast design using vibrant Orange and Cyan accents.
- **Interactive 3D Globe**: Custom procedural canvas-based globe in the Hero section.
- **Full-Area Garden Footer**: A unique, immersive SVG-based "Sunny Garden" footer with animated floating petals.
- **Dynamic Skills Bloom**: Animated "Bloom" circles for modern skill visualization.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop experiences.
- **Project Showcase**: Detailed project cards with modals and GitHub links.

## 🛠️ Tech Stack

- **Frontend**: React, CSS (Vanilla), Canvas 2D, SVG.
- **Design**: Glassmorphism, Premium Typography, Custom Micro-animations.
- **Backend (Showcased)**: Java (Servlets, JSP, JDBC), Python (OpenCV).

## 🚀 Getting Started

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/Rudra-Gupta15/Project_25.git
    ```
2.  **Install dependencies**:
    ```bash
    npm install
    ```
3.  **Run locally**:
    ```bash
    npm start
    ```

## 📂 Project Structure

- `src/components/`: Modular React components.
- `src/styles/`: Theme-based CSS modules.
- `public/`: Assets and static files.
